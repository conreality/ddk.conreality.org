##############################################
Conreality Driver Development Kit (DDK) Manual
##############################################

This is the device driver development reference manual for the `Conreality`_
live-action wargame.

The latest version of this manual can be browsed online at
`ddk.conreality.org`_, and it is also available for download in the
`PDF`_ and `EPUB`_ formats.

.. _Conreality:         https://conreality.org/
.. _ddk.conreality.org: https://ddk.conreality.org/
.. _PDF:                https://ddk.conreality.org/download/conreality-ddk.pdf
.. _EPUB:               https://ddk.conreality.org/download/conreality-ddk.epub
